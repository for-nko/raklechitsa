### Настройка сборки

```bash
# Делаем клон репозитория
$ https://github.com/for-nko/raklechitsa.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
